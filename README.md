# 廣岡 明登（Akito Hirooka）／ 個人プロモーションサイト

変革実装型プロジェクトリーダーとしての個人サイトです。**講演・研修・アドバイザリー（個人）** と、**行政・公共のDX/生成AI案件の受託相談（アデコ株式会社）** の2つの依頼導線を持ちます。GitHub Pages（ユーザーサイト）で公開します。

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 公開ページ本体（自己完結の静的HTML。外部依存はGoogle Fontsのみ） |
| `.nojekyll` | GitHub Pages の Jekyll 処理を無効化（静的HTMLをそのまま配信） |

## 公開手順（GitHub Pages）

1. このフォルダを GitHub の **`akitoh.github.io`** リポジトリへ push
2. リポジトリの **Settings → Pages**
3. **Source** を `Deploy from a branch`、Branch を `main` / `/(root)` に設定
4. 数分後 `https://akitoh.github.io/` で公開

> ユーザーサイト（`<ユーザー名>.github.io`）はリポジトリ名がそのまま公開URLになります。
